The MSDK includes example projects for each supported microcontroller. They demonstrate how to use the [peripheral driver APIs](/libraries-peripheral-drivers) and other supported libraries. Examples are located in the `Examples/<TARGET>/` folder of your MSDK installation.

<Warning>
  Copy example projects to a folder **outside** the MSDK installation before modifying them. This keeps the original copy intact for reference. The destination folder must not have any spaces in its filepath.
</Warning>

Each example contains a `README.md` describing what it does. In general, there is at least one example per peripheral block, and the example name matches the peripheral it demonstrates (for example, `DMA`, `ADC`, `SPI`).

## Available examples

The following examples are available for the MAX78000. Other supported targets have a similar set.

## Open an example

<Tabs>
  <Tab title="VS Code">

    <Steps>
      <Step title="Copy the example">
        Copy the example folder to a location outside the MSDK installation. The path must not contain spaces.
      </Step>
      <Step title="Open the folder">
        In VS Code, select **File > Open Folder** and navigate to the copied example folder.
      </Step>
      <Step title="Build and run">
        Use the VS Code build tasks to compile and flash the project. See the [VS Code IDE guide](/ide-vscode) for details.
      </Step>
    </Steps>

  </Tab>
  <Tab title="Eclipse">

    <Steps>
      <Step title="Copy the example">
        Copy the example folder to a location outside the MSDK installation. The path must not contain spaces.
      </Step>
      <Step title="Import the project">
        In Eclipse, select **File > Import > General > Existing Projects into Workspace**.
      </Step>
      <Step title="Select the folder">
        Set the root directory to the copied example folder and click **Finish**.
      </Step>
      <Step title="Build and run">
        Build the project using the Eclipse build tools. See the [Eclipse IDE guide](/ide-eclipse) for details.
      </Step>
    </Steps>

  </Tab>
  <Tab title="Command line">

    <Steps>
      <Step title="Copy the example">
        Copy the example folder to a location outside the MSDK installation. The path must not contain spaces.
      </Step>
      <Step title="Navigate to the folder">

        ```bash
        cd /path/to/copied/example
        ```

      </Step>
      <Step title="Build the project">

        ```bash
        make
        ```

      </Step>
      <Step title="Flash the binary">

        ```bash
        make flash
        ```

      </Step>
    </Steps>

  </Tab>
</Tabs>

<Tip>
  Always work from a copy of the example outside the MSDK folder. This lets you freely modify the project while keeping the original intact for reference.
</Tip>
